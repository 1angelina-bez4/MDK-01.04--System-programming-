# 📚  RU О курсе МДК 01.04

**МДК 01.04 «Системное программирование»** — это междисциплинарный курс в структуре программы подготовки специалистов среднего звена по специальности **09.02.07 «Информационные системы и программирование»** (квалификация — «Программист»).

Курс направлен на изучение низкоуровневого взаимодействия с аппаратным обеспечением, управление памятью, процессами и разработку эффективных алгоритмов.

### Что было изучено в рамках МДК 01.04:
- **⚙️ Ассемблер (встроенный в C++):** 10 лабораторных работ, посвященных регистрам, прямым командам процессора, оптимизации кода и взаимодействию ассемблерных вставок с языком высокого уровня (C++).
- **🔧 Язык C:** Работы, посвященные системным вызовам, работе с указателями, динамической памятью (`malloc`, `free`) и другим аспектам системного программирования на чистом C (работы в процессе добавления)

# EN 📚 About the MDK 01.04 Course

**MDK 01.04 "System Programming"** is an interdisciplinary course in the structure of the mid-level specialist training program for the specialty **09.02.07 "Information Systems and Programming"** (qualification — "Programmer").

The course focuses on studying low-level interaction with hardware, memory management, processes, and the development of efficient algorithms.

### What was studied within MDK 01.04:
- **⚙️ Assembly (embedded in C++):** 10 laboratory works dedicated to registers, direct processor instructions, code optimization, and interaction of assembly insertions with a high-level language (C++).
- **🔧 C Language:** Works dedicated to system calls, working with pointers, dynamic memory (`malloc`, `free`), and other aspects of system programming in pure C (works in the process of being added).

## 🚀 About the Repository

This repository is a portfolio of laboratory and practical works completed as part of the training in the specialty **09.02.07 "Information Systems and Programming"**. It contains projects demonstrating programming skills at different levels of abstraction: from direct interaction with the "hardware" to creating applications on modern platforms.
